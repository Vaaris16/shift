# Shift

## About
Allows users to set different wallpaper on there systems depending on the part of the day; morning, afternoon, and evening.

## Installation 

### macOS

1. Download the `.dmg` file from the **Releases** page.
2. Open the `.dmg` and drag **Shift** into the **Applications** folder.
3. Open **Shift** from Applications.

> ⚠️ **First launch warning (macOS)**  
> This app is not notarized by Apple yet. On first launch, macOS may show:  
> **“Shift is damaged and can’t be opened.”**
>
> To allow the app:
> - Open **System Settings → Privacy & Security**
> - Scroll down
> - Click **Open Anyway**
> - Confirm the dialog
>
> This is required only once.

---

### Windows

1. Download one of the following:
   - `shift_0.1.0_x64-setup.exe` (recommended)
   - `shift_0.1.0_x64_en-US.msi`
2. Run the installer.
3. Follow the setup instructions.

> If Windows SmartScreen appears, click **More info → Run anyway**.
